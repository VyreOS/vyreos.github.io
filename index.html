Welcome to VYRE OS

VYRE OS is a minimalist, performance-focused Linux distribution based on Debian. Designed for developers, gamers, and professionals.

Features

  • Based on Debian 12
  • Cinnamon Desktop (dark theme)
  • Steam and VS Code pre-installed
  • Calamares installer
