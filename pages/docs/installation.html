Installation Guide

1. Prepare Installation Media

sudo dd if=vyre-os-26.03-x86_64.iso of=/dev/sdX bs=4M status=progress

2. Boot from USB

  1. Insert USB drive
  2. Enter BIOS/UEFI (F2, F12, or Del)
  3. Set USB as first boot device
  4. Save and exit

3. Run Calamares Installer

  1. Select language
  2. Select timezone
  3. Configure partitions
  4. Create user
  5. Install
